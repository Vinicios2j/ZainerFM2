# 🎧 Zainer FM — Landing Page Profissional

Uma landing page moderna e interativa desenvolvida para rádios online, com foco em experiência do usuário, design premium e alta conversão.

---

## 🚀 Sobre o Projeto

O **Zainer FM** é um site institucional desenvolvido para rádios digitais que desejam fortalecer sua presença online, divulgar programação e atrair novos ouvintes.

Este projeto foi construído com foco em:

* 🎯 Conversão de visitantes em ouvintes
* 🎨 Design moderno e responsivo
* ⚡ Performance e leveza
* 📱 Experiência fluida em qualquer dispositivo

---

## ✨ Funcionalidades

* 🎧 Player flutuante (estrutura pronta para integração com rádio)
* 📅 Seção de programação dinâmica
* 📰 Área para notícias/blog
* 📢 Espaços para anúncios (monetização)
* 📱 Layout totalmente responsivo
* ⚡ Animações suaves com ScrollReveal
* 🎯 Botões estratégicos de ação (CTA)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* ScrollReveal.js

---

## 🧠 Diferenciais

* Interface moderna estilo rádio profissional
* Estrutura pronta para integração com streaming
* Ideal para rádios, podcasts ou mídia digital
* Código limpo e organizado para fácil manutenção

---

## 🔌 Integração com Rádio (Opcional)

O sistema já está preparado para integração com streaming ao vivo.

Para ativar:

1. Obter o link direto do stream da rádio (MP3 / AAC)
2. Inserir no elemento `<audio>` do projeto
3. Controlar via JavaScript (play/pause)
